# Grama-Khata 📒

**Your Digital Village Ledger**  
**ಗ್ರಾಮ-ಖಾತೆ | ನಿಮ್ಮ ಡಿಜಿಟಲ್ ವಹಿ**

A modern Android application built with **Kotlin** and **Jetpack Compose** for managing village-level accounts, customer ledgers, dues, and financial records — designed especially for Grama Panchayat use cases in Karnataka.

---

## ✨ Features

- **Beautiful Splash Screen** with Kannada & English support
- **Modern Material 3 UI** with smooth animations
- **Bottom Navigation** with 4 main sections:
  - **Home/Dashboard** – Quick overview and summary
  - **Customers** – Manage people and their ledgers
  - **Dues** – Track pending payments and reminders
  - **Reports** – Financial summaries and insights
- **Multi-language Support** (English + Kannada)
- **Customer Management** (Add, View, Transactions)
- **Credit / Debit Transactions**
- **Settings** screen for preferences

---

## 🛠 Tech Stack

- **Language**: Kotlin
- **UI**: Jetpack Compose
- **Architecture**: MVVM
- **Navigation**: Jetpack Navigation Compose
- **Build Tool**: Gradle (KTS)
